Introducción al Desarrollo Web

Este es un proyecto que integra los elementos fundamentales de HTML y CSS. A través de estos ejercicios se practica la estructura semántica y el diseño básico de una página.

Lo que aprenderás

  • Estructura con etiquetas semánticas.
  • Inclusión de multimedia (imágenes y video).
  • Creación de formularios y tablas.
  1. Configurar el entorno de desarrollo.
  2. Escribir código HTML válido.
  3. Aplicar estilos con CSS.
  4. Publicar en GitHub Pages.
