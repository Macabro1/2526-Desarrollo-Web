# Mi Sitio Web con Bootstrap 5 y CSS3 Avanzado

Este proyecto es una página web moderna, responsiva y estilizada creada como parte de una actividad académica para demostrar habilidades en CSS3 avanzado y Bootstrap 5.

## Características

- **Diseño completamente responsivo** que se adapta a móviles, tablets y escritorios
- **Barra de navegación fija** con efecto hover en los enlaces
- **Grid de Bootstrap** con 3 columnas en forma de cards interactivas
- **Formulario de contacto** con validación básica
- **Galería de imágenes** con efectos hover
- **Animaciones CSS3** personalizadas
- **Sistema de colores personalizado** que sobrescribe Bootstrap
- **Efectos de transición y transformación** en elementos interactivos

## Tecnologías Utilizadas

- HTML5 semántico
- CSS3 con animaciones, transiciones y media queries
- Bootstrap 5 (Grid, componentes, utilidades)
- Bootstrap Icons
- JavaScript básico para interactividad

## Personalizaciones CSS

Se han sobrescrito varios estilos de Bootstrap, incluyendo:

- Color primario personalizado (#ff5733)
- Efectos hover en cards y botones
- Animaciones de entrada para elementos
- Media queries personalizadas para pantallas pequeñas
- Sombreados y bordes redondeados personalizados

## Instrucciones de Despliegue

1. Subir todos los archivos a un repositorio de GitHub
2. Activar GitHub Pages en Settings > Pages
3. Seleccionar la rama main y la carpeta raíz
4. La página estará disponible en `https://[usuario].github.io/[repositorio]`
